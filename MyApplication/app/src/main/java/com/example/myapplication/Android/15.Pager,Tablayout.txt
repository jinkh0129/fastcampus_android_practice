

15. Pager / Tablayout
-->탭을 누르면 UI가 변하는 화면을 그릴 때 사용하는 장치들이다.
- 각각의 역할
    1. Pager ---> 안드로이드에서 제공을 함
     - '화면' : 종이를 넘기듯이 화면을 넘겨주는 역할
    2. Tablayout ---> 안드로이드에서 제공을 안하기 때문에 gradle을 수정해야한다.
     - '탭' : 탭을 담당하는 역할
    ---> 보통 같이 사용을 하지만 따로 사용을 해도 무방하다

    ==> 탭을 눌렀을 때 페이지가 넘어가기 위해서는 연결해주는 것이 필요하다
        : Adapter가 그 역할을 한다.
         - Tablayout과 Pager를 연결해주는 역할이다.
