

안드로이드에서 리스트를 그리는 방법
(리스트 --> 반복되는 비슷한 화면들)

***** 리스트뷰 *****
- 유사하게 반복되는 뷰를 그리기 위한 도구
    ex) 카카오톡의 채팅방들은 다 유사하게 생겼다

- 그리는 방법
    > addView --> 실제로 리스트뷰를 그리기 위해서 잘 사용되지 않음
        1. Item을 담을 xml을 만들어 준다
        2. 그 xml에 내용을 채워준다
        3. Container View에 더해준다
        4. 반복한다
    > ListView --> 예전에 많이 사용되었음
    > RecycleView --> 최근에 많이 사용이 되고 있고 가장 효율이 높다
